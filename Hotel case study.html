    Project Overview

    Crown is a responsive web template I designed specifically for the restaurant and food industry. It provides restaurant owners with a modern, user-friendly interface to engage customers effectively and build trust through a professional online presence.

    I drew inspiration from various industry-leading restaurant websites and platforms like Dribbble to ensure the design maximizes user experience and visual appeal, creating an intuitive and appealing journey for visitors.

    It has many other features like Crown repo contains a playground link which people can use to test the template with different theme colours to find their own preferred primary colour for the template.

    Feel free to check out the Project by visiting the Project Link.
